# Aplikasi Sejarah Android & Informasi Device

## Deskripsi Singkat
Aplikasi ini merupakan aplikasi Android sederhana yang terdiri dari dua halaman.
Halaman pertama menampilkan sejarah singkat perkembangan Android.
Halaman kedua menampilkan informasi perangkat (device) yang sedang digunakan seperti versi Android, model perangkat, brand, kernel, status baterai, dan status jaringan.

## Fitur Aplikasi
1. Menampilkan sejarah perkembangan Android.
2. Navigasi antar halaman menggunakan tombol.
3. Menampilkan informasi perangkat secara realtime.
4. Menampilkan status jaringan (WiFi / Cellular / Offline).
5. Menampilkan persentase baterai.

## Cara Menggunakan Aplikasi
1. Jalankan aplikasi.
2. Pada halaman pertama akan ditampilkan informasi sejarah Android.
3. Tekan tombol "Lihat Arsitektur & Info Device".
4. Aplikasi akan berpindah ke halaman kedua.
5. Informasi perangkat akan ditampilkan secara otomatis.
6. Tekan tombol kembali untuk kembali ke halaman sebelumnya.
